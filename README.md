# Kidney Disease Severity Classification Agent

## Project Overview

The **Kidney Disease Severity Classification Agent** is an AI-powered healthcare project designed to classify kidney disease severity levels using machine learning models and provide intelligent predictions based on patient health parameters.

This project combines:

* **Machine Learning models** for severity classification
* **AI Agent** for prediction and report generation
* **Web Application** using Streamlit/Flask for user interaction

The system helps healthcare professionals and users analyze kidney disease progression using patient clinical data.

---

## Objectives

* Predict kidney disease severity levels accurately
* Compare multiple machine learning models
* Generate intelligent health risk reports
* Track disease progression
* Maintain patient records for analysis

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Kidney-Disease-Severity-Classification-Agent.git
```

Move into project directory:

```bash
cd Kidney-Disease-Severity-Classification-Agent
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

OR

```bash
python app.py
```
